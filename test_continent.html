IICR functions for the continent-island model
¶

This is a test notebook for the Python package continent.

For an continent-island model with $n-1$ islands, $T_{k,\sigma}^{(K),n,p}$ denotes the $k$-th coalescence time where $K$ is the initial number of genes, $p$ the set of migration-coalescence parameters and $\sigma$ the initial state, i.e. an initial repartition of the $K$ genes in the continent and the $n-1$ islands.

The set of parameters has the form $p=[M_1,c_1,M_2,c_2]$ where

  • $M_1$ is the migration rate from the continent to the islands,
  • $c_1$ is the coalescence rate in the continent,
  • $M_2$ is the migration rate from the islands to the continent,
  • $c_2$ is the coalescence rate in the islands.

Denoting by $F(t)$ the cumulative distribution function $F(t) = \mbox{P}( T_{k,\sigma}^{(K),n,p} \leq t )$, the IICR is defined by $$ \lambda_{k,\sigma}^{(K),n,p}(t) = \frac{k(k-1)}{2} \; \frac{1 - F(t)}{F'(t)}. $$
