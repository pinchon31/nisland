IICR functions for the n-island model
¶

This is a test notebook for the Python package nisland.

For an n-island model with $n$ islands, $T_{k,\sigma}^{(K),n,M}$ denotes the $k$-th coalescence time where $K$ is the initial number of genes, $M$ the constant migration rate and $\sigma$ the initial state, i.e. an initial repartition of the $K$ genes in the $n$ islands.

Denoting by $F(t)$ the cumulative distribution function $F(t) = \mbox{P}_\sigma ( T_{k,\sigma}^{(K),n,M} \leq t )$, the IICR is defined by $$ \lambda_{k,\sigma}^{(K),n,M}(t) = \frac{k(k-1)}{2} \; \frac{1 - F(t)}{F'(t)}. $$
